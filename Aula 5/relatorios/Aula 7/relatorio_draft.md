# Relatório: Desenvolvimento da Landing Page GameTECH

## Resumo
O presente trabalho descreve o desenvolvimento de uma landing page para a empresa GameTECH, especializada na venda de jogos digitais, consoles e acessórios. A atividade teve como objetivo organizar uma interface web focada na experiência do usuário (UX), estruturando os produtos nas seções de Games, Utilidades e Acessórios. A página foi construída com tecnologias web padrão (HTML5, CSS3, e JavaScript) para garantir a exibição moderna dos anúncios por meio de cards interativos, incluindo a implementação de carrosséis de imagens tanto no banner principal de lançamento do Ark 2, quanto nos cards dos próprios produtos (Cadeira Snake Lancaster e Suporte BG500). As estratégias de visibilidade focaram em mitigar os problemas de estoque dos itens mencionados no briefing do projeto.

## Palavras Chaves
Landing Page, HTML5, CSS3, JavaScript, GameTECH.

## Abstract
The present work describes the development of a landing page for the company GameTECH, specialized in the sales of digital games, consoles, and accessories. The activity aimed to organize a web interface focused on User Experience (UX), structuring products into Games, Utilities, and Accessories sections. The page was built with standard web technologies (HTML5, CSS3, and JavaScript) to ensure the modern display of advertisements through interactive cards, including the implementation of image carousels on the Ark 2 main launch banner, as well as on the individual product cards (Snake Lancaster gaming chair and BG500 stand). Visibility strategies focused on mitigating stock issues for the items mentioned in the project briefing.

## Keywords
Landing Page, HTML5, CSS3, JavaScript, GameTECH.

## INTRODUÇÃO
A atividade proposta visa a criação de uma página de vendas (Landing Page) simulada para uma empresa do ramo de jogos chamada GameTECH. Foi solicitado um layout moderno e objetivo, dividindo a página em três categorias vitais: Games, Utilidades e Acessórios.
O principal objetivo não foi somente o desenvolvimento tecnológico, mas o planejamento de campanhas estratégicas de layout. A página necessitava atrair a viabilidade de vendas para a cadeira gamer Snake Lancaster (por conta da baixa procura) e a "queima de estoque" do suporte de notebooks Gaming BG500 (em alto volume). Por fim, o projeto devia destacar massivamente o lançamento do jogo Ark 2.

## METODOLOGIA
A execução contou com um planejamento lógico do design estrutural. As tarefas foram divididas e padronizadas da seguinte forma:
1. Montagem inicial do documento HTML com semântica para separar as áreas essenciais ("header", "nav", "sections", e "footer").
2. Estilização base via CSS Vanilla, adotando uma paleta de cores minimalista sob o tema Dark Mode (com contrastes entre cinza, ciano e vermelho), para agradar com sofisticação ao leitor que pertence à classe consumidora de alta tecnologia e jogos.
3. Aplicação da linguagem JavaScript para introduzir o funcionamento dinâmico com passagem iterativa de carrosséis fotográficos nas diferentes categorias do projeto com o intuito de valorizar visualmente os itens anunciados.

## MATERIAIS E MÉTODOS
Para chegar no resultado esperado e na construção fiel do código e documentação em PDF, foram utilizadas as seguintes ferramentas:
- **Editor de Texto IDE**: Para a produção primária e estruturação das sintaxes do web software.
- **HTML5**: Para o desenvolvimento semântico de blocos estruturais de mídia (Textos, Imagens, Tags).
- **CSS3**: Para aplicar toda a hierarquização estética web (ex.: responsividade via layout CSS Grid, margens, tamanhos, cores, posicionamento e hovers).
- **JavaScript (Vanilla JS)**: Para controlar o uso automatizado das galerias interativas simultaneamente, e a manipulação manual e abstrata da manipulação do DOM (Document Object Model).
- **Navegador Web**: Utilizado para os testes simultâneos de visibilidade responsiva do software.

## RESULTADOS E DISCUSSÃO
A página atingiu uma excelência estética e funcional condizente ao briefing que exigia uma postura direta de combate às concorrentes. Nas plataformas da Nuuvem e Green Man Gaming, geralmente, percebe-se um foco total na exibição exaustiva no catálogo de "chaves de jogo digitais", criando um poluído repositório de capas de título desde o primeiro contato. Na interface GameTECH desenvolvida, os produtos que precisavam de alavancagem de comércio exterior aos games (cadeira de descanso e suporte de refrigeração) ganharam alas seccionadas para eles com selos de indução imperativas (ex: "Queima de Estoque" e "Oferta Imperdível"). Esta estratégia atrelada a carrosséis descritivos com a fluidez simplificada na navegação otimizaram a perspectiva de marketing para giro do hardware enquanto atrai fortemente a atenção do leitor com exclusividades pelo topo com o lançamento ilustrativo do título Ark 2.

## CONCLUSÕES
O trabalho foi concluído satisfatoriamente evidenciando-se a conquista do briefing inicial do cliente. Os requisitos técnicos restritos à landing page foram superados e moldados para além do superficial, ofertando à página as garantias de boa acessibilidade via contrastes de elementos dinâmicos atrelados à estratégia de preços. Como dificuldade percebida, o maior desafio prático do layout evidenciou-se na responsabilidade univalente do JavaScript, onde foi necessário criar um algoritmo abstrato em que múltiplas instâncias de galerias em carrossel pelo site todo operassem sem corromper o estado das imagens dispostas em sub-módulos distantes na visualização geral.
